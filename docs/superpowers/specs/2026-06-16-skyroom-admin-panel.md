# IRoom — Skyroom-Style Admin Panel Design

**Date:** 2026-06-16  
**Type:** UI/UX Design Specification  
**Scope:** Complete admin panel redesign to match Skyroom's management interface

---

## 1. Design Philosophy

### Reference: Skyroom Admin Panel
Skyroom's admin panel (panel.skyroom.ir) is the gold standard for Iranian online classroom management. Key characteristics:

- **Dashboard-first:** Live overview with real-time stats, active rooms, and system health
- **Room-centric:** Rooms/classes are the primary entity — everything revolves around them
- **Tab-based navigation:** Clean tab switching between Users, Rooms, Sessions, Settings
- **Inline actions:** Edit, delete, view details without leaving the page
- **Modal-based forms:** Create/edit in modals, not separate pages
- **Status badges everywhere:** Live, scheduled, ended — color-coded at a glance
- **Bulk operations:** Select multiple items for batch actions
- **Search + filter:** Always-visible search with status/type filters

### Persian-First Mandate
**All numbers must use Persian (Eastern Arabic) numerals:** ۰۱۲۳۴۵۶۷۸۹  
**All dates must use Jalali (Shamsi) calendar**  
**All text must be in Farsi**  
**RTL layout throughout**

## 11. Persian-Only Mandate — Implementation Rules

### 11.1 Numbers
All numbers displayed in the UI **must** use Persian (Eastern Arabic) numerals:

| Digit | Persian |
|-------|---------|
| 0 | ۰ |
| 1 | ۱ |
| 2 | ۲ |
| 3 | ۳ |
| 4 | ۴ |
| 5 | ۵ |
| 6 | ۶ |
| 7 | ۷ |
| 8 | ۸ |
| 9 | ۹ |

**Implementation:** Create a utility function:
```typescript
// web/src/lib/utils/persian.ts
export function toPersianNum(n: number | string): string {
  const persianDigits = ['۰', '۱', '۲', '۳', '۴', '۵', '۶', '۷', '۸', '۹'];
  return String(n).replace(/[0-9]/g, (d) => persianDigits[parseInt(d)]);
}
```

**Usage in Svelte:**
```svelte
<span>{toPersianNum(stats.users)}</span>
<!-- Output: ۱۲۳ -->
```

**Exceptions:** Email addresses, URLs, phone numbers in `dir="ltr"` fields should keep Western digits.

### 11.2 Calendar
All dates **must** use Jalali (Shamsi) calendar:

**Implementation:** Use `date-fns-jalali`:
```typescript
import { format as jalaliFormat } from 'date-fns-jalali';
import { faIR } from 'date-fns-jalali/locale';

export function formatJalali(date: Date | string, formatStr: string = 'yyyy/MM/dd'): string {
  return jalaliFormat(new Date(date), formatStr, { locale: faIR });
}

export function formatJalaliTime(date: Date | string): string {
  return jalaliFormat(new Date(date), 'yyyy/MM/dd ساعت HH:mm', { locale: faIR });
}
```

**Date picker:** Replace all `<input type="date">` with a Jalali date picker component (e.g., `vue3-persian-datepicker` equivalent for Svelte, or custom component).

### 11.3 Text
- All UI strings must be in Farsi
- Use formal/polite forms (شما, لطفاً)
- Error messages in Farsi
- Success messages in Farsi
- Placeholder text in Farsi
